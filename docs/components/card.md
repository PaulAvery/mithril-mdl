# Cards
## Card
*mixin*

**Tag:** `div`  
**Class:** `mdl-card`

## CardTitle
**Tag:** `div`  
**Class:** `mdl-card__title`

**Attributes:**
* `expand`: Sets `mdl-card--expand`

A child `<h2>` element will be created which all children will be placed into.
